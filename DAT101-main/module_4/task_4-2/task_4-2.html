1. What is an array?

    2. An array is a collection of values stored in a single variable. Each value can be accessed using an index.

  2. What does it mean to "hardcode" when we talk about programming?

    3. "Hardcoding" means directly embedding values into code, rather than using variables or user input to dynamically determine them.

  6. In task 8, we will use a "callback" routine when sorting. Explain how such a "callback" routine works.

    7. A callback function in sorting is a comparison function passed to the sort method, determining the order of elements. It compares two values at a time and returns a number indicating their order, enabling customized sorting logic.

  7. What happens in the sorting routines you have created? Try to explain how the array can figure out if an element should come before or after another. The best way to answer this is to actually print both parameters each time the sorting routine runs.

    8. In the sorting routines, the callback function compares two elements, deciding which should come first by returning a positive or negative value. For ascending order, it returns a - b, and for descending, b - a, helping arrange elements in the desired order.
